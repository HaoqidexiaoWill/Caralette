### 利用集合存已经筛选过的

```python
class Solution:
    def findRepeatNumber(self, nums: List[int]) -> int:
        count = set()
        for i in nums:
            if i in count:
                return i
            else:
                count.add(i)
```

### 原地算法

```python
class Solution:
    def findRepeatNumber(self, nums: List[int]) -> int:
        for idx,val in enumerate(nums):
            if idx!=val and nums[val] == val:
                return val
            nums[val],nums[idx] = nums[idx],nums[val]
```
